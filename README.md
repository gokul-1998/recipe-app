# How to run the application

- clone the  repository
- navigate to the project directory
- install packages -> `npm install`
- to start the application ->`npm run dev`

# Features
- Create a new recipe
- View all recipes
- View a single recipe
- Delete a recipe


# Technologies
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- tailwindcss
